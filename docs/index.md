# 欢迎使用 Markdown 文档网站

这是一个基于 PHP 的 Markdown 文档系统，可以将文件夹中的 Markdown 文档自动转换为美观的网站。

## 主要特性

- 📁 **文件夹组织** - 按照文件夹结构自动生成目录导航
- 🎨 **现代设计** - 简洁美观的界面设计
- 🔍 **全文搜索** - 支持实时搜索文档内容
- 💻 **代码高亮** - 支持多种编程语言的语法高亮
- 📐 **数学公式** - 支持 LaTeX 数学公式渲染
- 📱 **响应式布局** - 适配桌面和移动设备
- ⚡ **缓存机制** - 目录树缓存，自动更新

## 快速开始

访问左侧目录中的 **入门指南** 开始使用。
